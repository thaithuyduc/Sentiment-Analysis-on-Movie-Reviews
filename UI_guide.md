To open UI for this project, follow these steps
1. Get Youtube API key
    1. Access to Google Cloud Console: https://console.cloud.google.com/
    2. Create new Project (or choose available Project)
    3. Navigate to APIs & Services → Library
    4. Enable “YouTube Data API v3”
    5. Naviage to APIs & Services → Credentials
    6. Choose Create Credentials → API Key. Then copy this API Key
2. Navigate to `app2.py`
3. Paste API Key in `API_KEY = `
4. Open terminal and run `streamlit run app2.py` in terminal
